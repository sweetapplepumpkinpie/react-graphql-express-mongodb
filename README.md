# MERNG stack (MongoDB, Express, React JS, Node and GraphQL)

### Technolgies used

* Node JS
* Express
* Apollo Server
* GraphQL
* Mongoose
* Mongo DB
* React JS
* Prime React
* Apollo Client
* Formik

### Project features

* Register user
* Login user
* View all posts
* View a single post
* Create a post
* Delete owned posts
* Comment on a post
* Delete owned comments
* Like and unlike posts
